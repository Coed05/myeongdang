등급 범례

사용 방법

  1. 주소 입력 후 후보 선택
  2. 반경·주야·가중치 조정(선택)
  3. 안심 등급 분석 클릭
  4. 지도에서 주변 공장·바람 확인

데이터

  • 공장·업종 — 한국산업단지공단
  • 주소·좌표 — 카카오 로컬 · 브이월드 · OSM
  • 기상 — 기상청 단기예보

참고

정부 공인 배출 원단위·소음 표준과 지역 기상통계를 화공 수식에 적용한 추정치예요. 실제 환경은 기상·운영 상황에 따라 달라질 수 있어요.
