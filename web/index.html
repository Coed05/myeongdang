주소만 입력하면 인근 산업단지를 자동 탐지해 소음·악취 안심 등급(AAA~F)을 매겨요.
📚 분석 방법론 · 과학적 근거

왜 시뮬레이션인가? 공장의 실시간 배출량은 영업비밀이라 공개되지 않아요. 정부 공인 '배출 원단위 가중치'와 '물리적 소음 표준'을 공인 화공 수식에 대입해요.

3대 핵심 배출 업종 — 화학(C20) Q=100·85dB / 고무·플라스틱(C22) 60·80dB / 금속가공(C25) 15·95dB

악취 — 2차원 가우스 확산 $C = \dfrac{Q}{\pi\,u\,\sigma_y\,\sigma_z}\exp\!\left(-\dfrac{y^2}{2\sigma_y^2}\right)\exp\!\left(-\dfrac{H^2}{2\sigma_z^2}\right)$,   $\sigma_y=a\,x^{b},\ \sigma_z=c\,x^{d}$ (Martin, 낮 C / 밤 E)

소음 — 거리 역자승 감쇄 $SPL_2=SPL_1-20\log_{10}(r_2)$ + 로그 합산

근거: 악취방지법, 산업안전보건기준 규칙 §512, Pasquill(1961), Martin(1976).
